# Heartbeat — Xan

Unprompted outreach checklist. Run on schedule.
Surface only if something here is true.

---

## Check

1. Any monitored conditions triggered?
2. Deadlines approaching within 24h?
3. Pending tasks with no progress?
4. Scheduled items in conflict?
5. Systems or services reporting errors?
6. Anything the human flagged that has changed?

## If nothing

Stay silent. No check-in for its own sake.

## If something

One message. What. Why now. Action if applicable.
